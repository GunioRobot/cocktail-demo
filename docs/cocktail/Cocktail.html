Cocktail()constructor
public function Cocktail(app:Sprite, embeder:EmbederTail, appId:String, defaultUrl:String, logLevel:uint = 5, logDetail:uint = 1)

Creates a new Cocktail instance.

Parameters
app:Sprite — The application reference.
 
embeder:EmbederTail — Application identifier (MUST be the the application folder name).
 
appId:String — Application default url.
 
defaultUrl:String — Application default log level (default=5).
# 0=disable
# 1=fatal
# 2=fatal,error
# 3=fatal,error,debug
# 4=fatal,error,debug,warn
# 5=fatal,error,debug,warn,info
 
logLevel:uint (default = 5) — Application default log detail (default=1).
# 0=Doesnt add any extra prefix besides the log level.
# 1=Adds a prefix to all log calls, with the 'ClassName'
# 2=Adds a prefix to all log calls, with the 'packace.folderX.folderY.ClassName'.
 
logDetail:uint (default = 1)
