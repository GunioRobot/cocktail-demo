Packagecocktail.core.gunz
Classpublic class Gunz

The Gunz class is a list of Gun's. Here you can remove all Gun listeners with rm_all and make recursive destruction of everything with destroy.

The way to add a Gun to this list is by passing this Gunz reference to the Gun constructor.

Examples

Creating a Gunz and associating Gun's in it. 

var gunz : Gunz = new Gunz( this );
  
var gun1 : Gun = new Gun( gunz, this, "type1" );
var gun2 : Gun = new Gun( gunz, this, "type2" );
var gun3 : Gun = new Gun( gunz, this, "type3" );
